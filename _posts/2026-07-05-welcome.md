这是一篇示例文章。

以后你只需要在 `_posts/` 目录新增 Markdown 文件，文件名使用 `YYYY-MM-DD-title.md`，并在文件顶部保留类似下面的 Front Matter：

```yaml
---
title: "文章标题"
date: 2026-07-05 12:00:00 +0800
categories: [tech]
tags: [example]
---
```

GitHub Pages 会在发布时自动把 Markdown 渲染成网页。

## 支持的内容

- 标题、列表、引用、表格
- 代码块和语法高亮
- 图片和链接
- 分类、标签、RSS 和 sitemap

```js
const message = "Write once in Markdown, publish with GitHub Pages.";
console.log(message);
```
